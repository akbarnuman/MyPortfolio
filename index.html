Who I am

I'm a Computer Science graduate from Lovely Professional University, with a habit of building things end-to-end — frontend, backend, and everything that has to talk to everything else in between. I care about writing software that's scalable and actually solves the problem in front of a user, not just technically correct.

My work leans toward real-time systems and AI-integrated applications: chat platforms with live typing indicators and file sharing, or web apps that call a TensorFlow model to diagnose a mushroom from a photo. I like the parts of engineering where a REST API has to coordinate cleanly with a websocket, a database, and a model service without anything falling over.

Right now I'm sharpening my Data Structures & Algorithms skills through competitive programming in Java — debugging greedy logic, reaching for the right technique (two pointers, Manacher's algorithm) instead of the first one that compiles — while actively looking for Software Engineer and Full-Stack Developer roles. Always up for learning something new or building alongside people who care about the details.
